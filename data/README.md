# Data

Dataset documentation for the Open Stethoscope project.

## Purpose

This folder will document recording metadata, dataset structure, labeling conventions, and links to released data.

## Planned Contents

- Metadata CSV schema
- Recording protocol
- Dataset organization notes
- ICBHI reference audio notes
- Zenodo release information

## Note

Large audio files will not be stored directly in this repository. Final datasets will be released through Zenodo or another research data archive.
